Description

NetHunter is a remake of the famous text-mode Linux game, Hunt.
You are a player trapped in a maze.
Your purpose? Kill 'em all !!!
Your enemies? Anyone from the internet willing to have some fun.
Your toys? Plasma gun + Fusion Bomb :)

NetHunter is designed to run even on slow computers using a modem.

Highlights

- Real-time internet game
- Slow computer & low bandwith internet connection
- Huge random generated maze
- Cool graphics
- Fog of war
- Chat console
- Ammo/health crates
- Stupid bots (for now) for single player mode
- Administrator user
- GUARANTEED FUN

Technical Highlights

- Client-Server architecture
- Server portable on Solaris, Linux and Windows
- TCP/UDP communication
- Direct Draw
- FMOD for sound/music
- Incorporates a custom MP3 player
