Description

Battle in amazing landscapes, battle in the day, in the night, winter, summer......anytime.
There is no way you can escape the challenge.
Bang Bang 3D is apparently a simple game where two players have to destroy each other using their cannon
But things are not so easy, because the wind comes and ruins everything

Bang Bang 3D requires a decent 3D graphics card for a maximum entertainment.

Highlights

- Real-time Day/Night alternation
- Two seasons: winter and summer
- It's snowing in winter
- Customizable terrain
- Custom MP3 player using Winamp playlists
- Particle effects
- Fog
- Animated Sky and Water
- Extremely detailed terrain
- Real physics

Technical Highlights

- OpenGL based
- Real-time 3D terrain generation using a heightmap
- Terrain rendering optimized using LOD
- Detail texturing using Multitexturing (only on video cards that support it)
- Frustum culling
- Uses FMOD for sound/music
- Models are created and exported from 3D Studio MAX using an export plugin
- Particle system that allows stretching
- Billboarding for particle rendering
